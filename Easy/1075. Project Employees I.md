## 🚀 1075. Project Employees I

### 💡 Idea

The goal is to find the **average years of experience** of employees working on each project.

The `Project` table maps employees to projects, while the `Employee` table contains each employee's experience. We join the two tables using `employee_id`, group the results by `project_id`, and calculate the average experience.

---

### 💻 Solution

```sql
SELECT p.project_id, ROUND(AVG(e.experience_years), 2) AS average_years
FROM Project p
INNER JOIN Employee e
ON p.employee_id = e.employee_id
GROUP BY p.project_id;
```

---

### 🛠️ How It Works

1. Join the `Project` and `Employee` tables using `employee_id`.
2. Group the records by `project_id`.
3. Calculate the average `experience_years` for employees in each project using `AVG()`.
4. Round the average to **2 decimal places** using `ROUND()`.
5. Return the project ID and the average years of experience.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single join followed by grouping)*

**Space Complexity:** `O(n)` *(stores grouped project records)*
