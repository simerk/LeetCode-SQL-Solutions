## 🚀 1741. Find Total Time Spent by Each Employee

### 💡 Idea

The goal is to calculate the **total time spent** by each employee on each day.

For every employee and event date:

- Compute the duration of each task as:
  - `out_time - in_time`
- Sum the durations of all tasks performed on that day.

Finally, return the employee ID, event day, and total time spent.

---

## 💻 Solution

```sql
SELECT event_day AS day, emp_id,
SUM(out_time - in_time) AS total_time
FROM Employees
GROUP BY event_day, emp_id;
```

---

### 🛠️ How It Works

1. Group the records by `event_day` and `emp_id`.
2. Calculate the duration of each task using:
   - `out_time - in_time`
3. Sum the durations of all tasks performed by the same employee on the same day.
4. Return the day, employee ID, and total time spent.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan followed by grouping)*

**Space Complexity:** `O(n)` *(stores grouped employee-day combinations)*
