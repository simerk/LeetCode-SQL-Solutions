## 🚀 1661. Average Time of Process per Machine

### 💡 Idea

The goal is to calculate the **average processing time** for each machine.

Each process consists of two records:

- `start` — when the process begins.
- `end` — when the process finishes.

The processing time for a process is:

```text
end timestamp - start timestamp
```

We can solve this problem using either a **Self Join** or **Conditional Aggregation**.

---

## 💻 Approach 1: Self Join

```sql
SELECT s.machine_id, ROUND(AVG(e.timestamp - s.timestamp), 3) AS processing_time
FROM Activity s
JOIN Activity e 
ON s.machine_id = e.machine_id
AND s.process_id = e.process_id
WHERE s.activity_type = 'start' AND e.activity_type = 'end'
GROUP BY s.machine_id;
```

### 🛠️ How It Works

1. Treat the `Activity` table as two separate tables:
   - `s` for **start** records.
   - `e` for **end** records.
2. Join them using `machine_id` and `process_id`.
3. Pair each `start` record with its corresponding `end` record.
4. Calculate the processing time using:
   - `e.timestamp - s.timestamp`
5. Compute the average processing time for each machine.
6. Round the result to **3 decimal places**.

---

## 💻 Approach 2: Conditional Aggregation

```sql
SELECT machine_id, ROUND(AVG(processing_time), 3) AS processing_time
FROM (
    SELECT machine_id, process_id,
    SUM(
        CASE
            WHEN activity_type = 'end' THEN timestamp
            ELSE -timestamp
        END ) AS processing_time
    FROM Activity
    GROUP BY machine_id, process_id
) t
GROUP BY machine_id;
```

### 🛠️ How It Works

1. Group the records by `machine_id` and `process_id`.
2. Convert:
   - `start` timestamps into negative values.
   - `end` timestamps into positive values.
3. Sum these values to obtain:
   - `end - start` for each process.
4. Group the results by `machine_id`.
5. Calculate the average processing time and round it to **3 decimal places**.

---

### ⏱️ Complexity

| Approach | Time | Space |
|----------|------|-------|
| Self Join | `O(n)` | `O(n)` |
| Conditional Aggregation | `O(n)` | `O(n)` |
