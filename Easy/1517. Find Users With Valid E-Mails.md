## 🚀 1517. Find Users With Valid E-Mails

### 💡 Idea

The goal is to return only the users whose email addresses are **valid** according to the problem's rules.

A valid email must:

- Start with a letter (`A-Z` or `a-z`).
- Contain only letters, digits, underscores (`_`), periods (`.`), or hyphens (`-`) before the `@`.
- End with the domain **`@leetcode.com`** in lowercase.

We use MySQL's `REGEXP` operator to validate the email format and `LIKE BINARY` to ensure the domain matches exactly in lowercase.

---

## 💻 Solution

```sql
SELECT *
FROM Users
WHERE mail REGEXP '^[A-Za-z][A-Za-z0-9_.-]*@leetcode\\.com$'
AND mail LIKE BINARY '%@leetcode.com';
```

---

### 🛠️ How It Works

1. `REGEXP` validates the overall email format.
2. `^` ensures the match starts at the beginning of the string.
3. `[A-Za-z]` requires the first character to be a letter.
4. `[A-Za-z0-9_.-]*` allows zero or more letters, digits, underscores (`_`), periods (`.`), or hyphens (`-`) before the `@`.
5. `@leetcode\\.com$` ensures the email ends with `@leetcode.com`.
6. `LIKE BINARY '%@leetcode.com'` performs a **case-sensitive** check, ensuring the domain is exactly `@leetcode.com` in lowercase.

---

### ⏱️ Complexity

**Time Complexity:** `O(n × m)`  
*(`n` = number of rows, `m` = average email length)*

**Space Complexity:** `O(1)`
