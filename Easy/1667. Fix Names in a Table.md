## 🚀 1667. Fix Names in a Table

### 💡 Idea

The goal is to format each user's name so that:

- The **first character** is uppercase.
- All **remaining characters** are lowercase.

We can achieve this by combining MySQL string functions:
- `UPPER()` to capitalize the first letter.
- `LOWER()` to convert the remaining letters to lowercase.
- `LEFT()` and `SUBSTRING()` to split the string.

Finally, sort the result by `user_id`.

---

## 💻 Solution

```sql
SELECT user_id,
CONCAT(
        UPPER(LEFT(name, 1)),
        LOWER(SUBSTRING(name, 2))
      ) AS name
FROM Users
ORDER BY user_id;
```

---

### 🛠️ How It Works

1. Extract the first character using `LEFT(name, 1)`.
2. Convert it to uppercase using `UPPER()`.
3. Extract the remaining characters using `SUBSTRING(name, 2)`.
4. Convert the remaining characters to lowercase using `LOWER()`.
5. Concatenate both parts using `CONCAT()`.
6. Return the formatted name ordered by `user_id`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n × m)`  
*(`n` = number of users, `m` = average length of a name)*

**Space Complexity:** `O(1)`
