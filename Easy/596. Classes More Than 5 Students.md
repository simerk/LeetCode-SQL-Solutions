## 🚀 596. Classes More Than 5 Students

### 💡 Idea

The goal is to find classes that have **at least 5 students**.

Group the records by `class`, count the number of students in each class, and use the `HAVING` clause to keep only those classes with a count of **5 or more**.

---

### 💻 Solution

```sql
SELECT class
FROM Courses
GROUP BY class
HAVING COUNT(*) >= 5;
```

---

### 🛠️ How It Works

1. Group all records by the `class` column.
2. Count the number of students in each class using `COUNT(*)`.
3. Use the `HAVING` clause to filter classes with at least **5** students.
4. Return the qualifying class names.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan with grouping, depending on indexing)*

**Space Complexity:** `O(n)` *(stores one grouped record per class)*
