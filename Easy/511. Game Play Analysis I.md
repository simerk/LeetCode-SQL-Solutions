## 🚀 511. Game Play Analysis I

### 💡 Idea

We need to find the **first login date** for each player.

Since the first login corresponds to the **earliest (`MIN`) `event_date`** for every player, we can group the records by `player_id` and use the `MIN()` aggregate function to retrieve the earliest date.

---

### 💻 Solution

```sql
SELECT player_id, MIN(event_date) AS first_login
FROM Activity
GROUP BY player_id
ORDER BY player_id;
```

---

### 🛠️ How It Works

1. Group all records by `player_id`.
2. For each player, find the earliest `event_date` using the `MIN()` function.
3. Rename the result as `first_login`.
4. Sort the output by `player_id`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan with grouping, depending on indexing)*

**Space Complexity:** `O(n)` *(stores one grouped result per player)*
