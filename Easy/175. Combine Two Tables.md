## 🚀 175. Combine Two Tables

### 💡 Idea

We need to display **every person** from the `Person` DataFrame, along with their address details if they exist.

We can use the `merge()` function with `how="left"` to perform a **left join**, ensuring that all rows from the `Person` DataFrame are included. If a person doesn't have a matching address, Pandas automatically fills the `city` and `state` columns with `NaN`, which represents missing values.

---

### 💻 Solution

```python
import pandas as pd

def combine_two_tables(person: pd.DataFrame, address: pd.DataFrame) -> pd.DataFrame:
    return person.merge(address, on="personId", how="left")[
        ["firstName", "lastName", "city", "state"]
    ]
```

---

### 🛠️ How It Works

1. Start with the **Person** DataFrame, which contains all people.
2. Merge it with the **Address** DataFrame using the common column `personId`.
3. Keep all rows from the `Person` DataFrame by performing a **left join**.
4. If a matching address exists, the corresponding `city` and `state` values are included.
5. If no matching address is found, the `city` and `state` columns contain `NaN`.
6. Finally, select only the required columns: `firstName`, `lastName`, `city`, and `state`.

---

### 📚 Pandas Functions Used

#### `merge()`

The `merge()` function combines two DataFrames based on one or more common columns, similar to SQL joins.

**Syntax**

```python
DataFrame.merge(right, on=None, how="inner")
```

**Parameters used:**

- `right` → The DataFrame to merge with.
- `on="personId"` → Specifies the common column used to match rows.
- `how="left"` → Keeps all rows from the left DataFrame and adds matching rows from the right DataFrame.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(merging the DataFrames based on `personId`)*

**Space Complexity:** `O(n)` *(stores the merged DataFrame containing the joined result)*
