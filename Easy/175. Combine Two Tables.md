## 🚀 176. Second Highest Salary

### 💡 Idea

We need to find the **second highest distinct salary** in the `Employee` table.

First, sort the distinct salaries in descending order. Then, skip the highest salary using `OFFSET 1` and retrieve the next one using `LIMIT 1`. If a second highest salary doesn't exist, return `NULL` using `IFNULL()`.

---

### 💻 Solution

```sql
SELECT IFNULL(
    (
        SELECT DISTINCT salary
        FROM Employee
        ORDER BY salary DESC
        LIMIT 1 OFFSET 1
    ), NULL
) AS SecondHighestSalary;
```

---

### 🛠️ How It Works

1. Select all **distinct** salaries to ignore duplicate salary values.
2. Sort the salaries in descending order.
3. Skip the highest salary using `OFFSET 1`.
4. Retrieve the next salary using `LIMIT 1`, which is the second highest distinct salary.
5. If no second highest salary exists, `IFNULL()` returns `NULL`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)` *(sorting the distinct salary values)*

**Space Complexity:** `O(n)` *(stores the distinct salary values during sorting)*
