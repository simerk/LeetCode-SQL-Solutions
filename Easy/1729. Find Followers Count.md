## 🚀 1729. Find Followers Count

### 💡 Idea

The goal is to find the number of followers for each user.

Each row in the `Followers` table represents one follower-followee relationship. We group the records by `user_id` and count the number of followers associated with each user.

Finally, the results are sorted by `user_id`.

---

## 💻 Solution

```sql
SELECT user_id, COUNT(follower_id) AS followers_count
FROM Followers
GROUP BY user_id
ORDER BY user_id;
```

---

### 🛠️ How It Works

1. Group the records by `user_id`.
2. Count the number of `follower_id` values for each user using `COUNT()`.
3. Return the user ID along with its follower count.
4. Sort the results by `user_id` in ascending order.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan followed by grouping)*

**Space Complexity:** `O(n)` *(stores grouped users)*
