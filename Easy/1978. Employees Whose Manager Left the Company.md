## 🚀 1978. Employees Whose Manager Left the Company

### 💡 Idea

The goal is to find employees who satisfy all of the following conditions:

- Their salary is **less than 30000**.
- They have a manager (`manager_id IS NOT NULL`).
- Their manager no longer exists in the `Employees` table.

We can use a subquery to check whether the manager's `employee_id` exists in the table.

---

## 💻 Solution

```sql
SELECT employee_id
FROM Employees
WHERE salary < 30000
  AND manager_id IS NOT NULL
  AND manager_id NOT IN (
      SELECT employee_id
      FROM Employees
  )
ORDER BY employee_id;
```

---

### 🛠️ How It Works

1. Filter employees whose salary is less than **30000**.
2. Exclude employees without a manager using `manager_id IS NOT NULL`.
3. Use a subquery to retrieve all existing employee IDs.
4. Keep only those employees whose `manager_id` is **not** present in that list.
5. Sort the result by `employee_id`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)`

**Space Complexity:** `O(n)`
