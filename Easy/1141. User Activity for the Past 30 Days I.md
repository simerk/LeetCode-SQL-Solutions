## 🚀 1141. User Activity for the Past 30 Days I

### 💡 Idea

The goal is to report the **number of active users for each day** during the 30-day period ending on **2019-07-27**.

A user is considered **active** if they performed at least one activity on that day. Therefore, for each date, we count the number of **distinct users** who were active.

---

### 💻 Solution

```sql
SELECT activity_date AS day, COUNT(DISTINCT user_id) AS active_users
FROM Activity
WHERE activity_date BETWEEN '2019-06-28' AND '2019-07-27'
GROUP BY activity_date;
```

---

### 🛠️ How It Works

1. Filter the records to include only activities between **2019-06-28** and **2019-07-27**.
2. Group the records by `activity_date`.
3. Count the number of **distinct** `user_id`s for each day.
4. Return the activity date as `day` and the corresponding number of active users.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan with filtering and grouping)*

**Space Complexity:** `O(n)` *(stores grouped activity dates)*
