## 🚀 595. Big Countries

### 💡 Idea

A country is considered **big** if **either** its area is at least **3,000,000 km²** **or** its population is at least **25,000,000**.

We simply filter the `World` table using these conditions and return the required columns.

---

### 💻 Solution

```sql
SELECT name, population, area
FROM World
WHERE area >= 3000000 OR population >= 25000000;
```

---

### 🛠️ How It Works

1. Select the `name`, `population`, and `area` columns from the `World` table.
2. Check if a country's `area` is greater than or equal to **3,000,000**.
3. Also check if its `population` is greater than or equal to **25,000,000**.
4. Use the `OR` operator so that countries satisfying **either** condition are included.
5. Return the resulting list of big countries.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan of the `World` table)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
