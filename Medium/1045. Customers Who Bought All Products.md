## 🚀 1045. Customers Who Bought All Products

### 💡 Idea

The goal is to find customers who have purchased **every product** listed in the `Product` table.

For each customer, count the number of **distinct products** they have bought. If this count matches the total number of products available, then the customer has purchased all products.

---

### 💻 Solution

```sql
SELECT customer_id
FROM Customer
GROUP BY customer_id
HAVING COUNT(DISTINCT product_key) = (
    SELECT COUNT(*)
    FROM Product
);
```

---

### 🛠️ How It Works

1. Group the records in the `Customer` table by `customer_id`.
2. Count the number of **distinct** products purchased by each customer.
3. Count the total number of products in the `Product` table.
4. Compare the two counts using the `HAVING` clause.
5. Return the customers whose product count matches the total number of products.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan of the `Customer` table with grouping, plus one scan of the `Product` table)*

**Space Complexity:** `O(n)` *(stores grouped customer records)*
