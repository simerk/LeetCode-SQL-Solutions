## 🚀 3421. Find Students Who Improved

### 💡 Idea

The goal is to find students whose **latest exam score** is higher than their **first exam score** for each subject.

We use the `ROW_NUMBER()` window function twice:

- Once to identify the **first exam**.
- Once to identify the **latest exam**.

Finally, we join both results and keep only the students whose latest score is greater than their first score.

---

## 💻 Solution

```sql
WITH first_exam AS (
    SELECT student_id, subject, score,
    ROW_NUMBER() OVER (
            PARTITION BY student_id, subject
            ORDER BY exam_date
        ) AS rn
    FROM Scores
),

latest_exam AS (
    SELECT student_id, subject, score,
    ROW_NUMBER() OVER (
            PARTITION BY student_id, subject
            ORDER BY exam_date DESC
        ) AS rn
    FROM Scores
),

SELECT f.student_id, f.subject,
f.score AS first_score, l.score AS latest_score
FROM first_exam f
JOIN latest_exam l
ON f.student_id = l.student_id AND f.subject = l.subject
WHERE f.rn = 1 AND l.rn = 1 AND l.score > f.score
ORDER BY f.student_id, f.subject;
```

---

### 🛠️ How It Works

1. Use `ROW_NUMBER()` to identify the **first exam** for each student and subject.
2. Use another `ROW_NUMBER()` to identify the **latest exam**.
3. Join both CTEs on `student_id` and `subject`.
4. Keep only the rows where the latest score is greater than the first score.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)`

**Space Complexity:** `O(n)`
