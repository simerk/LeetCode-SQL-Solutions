## 🚀 176. Second Highest Salary

### 💡 Idea

We need to find the **second highest distinct salary** in the `Employee` DataFrame.

First, remove duplicate salary values, then sort them in descending order. If there are at least two distinct salaries, return the second highest; otherwise, return `None`, which is displayed as `NULL` by LeetCode.

---

### 💻 Solution

```python
import pandas as pd

def second_highest_salary(employee: pd.DataFrame) -> pd.DataFrame:
    salaries = employee["salary"].drop_duplicates().sort_values(ascending=False)

    if len(salaries) < 2:
        return pd.DataFrame({"SecondHighestSalary": [None]})

    return pd.DataFrame({"SecondHighestSalary": [salaries.iloc[1]]})
```

---

### 🛠️ How It Works

1. Select the `salary` column from the **Employee** DataFrame.
2. Remove duplicate salary values using `drop_duplicates()`.
3. Sort the remaining salaries in descending order.
4. Check if there are at least two distinct salaries.
5. If not, return `None`.
6. Otherwise, retrieve the second salary using `iloc[1]`.
7. Return the result as a DataFrame with the column name `SecondHighestSalary`.

---

### 📚 Pandas Functions Used

#### `drop_duplicates()`

Removes duplicate values from a Series or DataFrame.

**Syntax**

```python
Series.drop_duplicates()
```

In this solution, it ensures that duplicate salary values are ignored.

---

#### `sort_values()`

Sorts the values of a Series or DataFrame.

**Syntax**

```python
Series.sort_values(ascending=True)
```

**Parameter used:**

- `ascending=False` → Sorts the salaries from highest to lowest.

---

#### `iloc[]`

Accesses rows or values by their integer position.

**Syntax**

```python
Series.iloc[index]
```

In this solution:

- `iloc[1]` returns the **second** highest distinct salary.

---

#### `pd.DataFrame()`

Creates a new DataFrame.

**Syntax**

```python
pd.DataFrame(data)
```

Here, it is used to return the result in the format expected by LeetCode.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)` *(sorting the distinct salary values)*

**Space Complexity:** `O(n)` *(stores the distinct salary values after removing duplicates)*
