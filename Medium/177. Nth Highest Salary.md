## 🚀 177. Nth Highest Salary

### 💡 Idea

The goal is to return the **Nth highest distinct salary** from the `Employee` table.

Since the `LIMIT` clause uses **zero-based indexing**, we first subtract `1` from `N`. Then, we sort the distinct salaries in descending order and retrieve the salary at the required position.

---

### 💻 Solution

```sql
CREATE FUNCTION getNthHighestSalary(N INT) RETURNS INT
BEGIN
    SET N = N - 1;
    RETURN (
        SELECT DISTINCT salary
        FROM Employee
        ORDER BY salary DESC
        LIMIT N, 1
    );
END
```

---

### 🛠️ How It Works

1. The function accepts an integer `N`.
2. Subtract `1` from `N` because `LIMIT` uses zero-based indexing.
3. Select only **distinct** salaries to ignore duplicates.
4. Sort the salaries in descending order.
5. Use `LIMIT N, 1` to return the salary at the desired position.
6. If the Nth highest salary doesn't exist, the function automatically returns `NULL`.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)` *(sorting the distinct salary values)*

**Space Complexity:** `O(n)` *(stores the distinct salary values during sorting)*
