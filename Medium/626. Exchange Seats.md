## 🚀 626. Exchange Seats

### 💡 Idea

The goal is to **swap the seat IDs of every two consecutive students**.

- If the seat ID is **odd**, swap it with the next seat.
- If the seat ID is **even**, swap it with the previous seat.
- If the last seat is **odd** and has no partner, keep it unchanged.

We can achieve this using a `CASE` expression to determine each student's new seat ID and then sort the result by the new ID.

---

### 💻 Solution

```sql
SELECT
    CASE
        WHEN MOD(id, 2) != 0 AND id != (SELECT COUNT(*) FROM Seat) THEN id + 1
        WHEN MOD(id, 2) = 0 THEN id - 1
        ELSE id
    END AS id,
    student
FROM Seat
ORDER BY id;
```

---

### 🛠️ How It Works

1. Check whether the current seat ID is odd or even using `MOD(id, 2)`.
2. If the seat ID is odd and it is **not the last seat**, increase the ID by `1`.
3. If the seat ID is even, decrease the ID by `1`.
4. If the seat is the last odd seat with no partner, leave its ID unchanged.
5. Return the updated seat IDs along with the student names.
6. Sort the final result by the new seat ID.

---

### ⏱️ Complexity

**Time Complexity:** `O(n)` *(single scan of the table)*

**Space Complexity:** `O(1)` *(excluding the output result set)*
