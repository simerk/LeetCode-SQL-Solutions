## 🚀 178. Rank Scores

### 💡 Idea

The goal is to rank scores in **descending order** while assigning the **same rank to duplicate scores** without leaving any gaps in the ranking.

The `DENSE_RANK()` window function is perfect for this because it generates consecutive ranks even when multiple rows have the same score.

---

### 💻 Solution

```sql
SELECT score, DENSE_RANK() OVER (ORDER BY score DESC) AS `rank`
FROM Scores
ORDER BY score DESC;
```

---

### 🛠️ How It Works

1. Sort all scores in descending order using `ORDER BY score DESC`.
2. Apply the `DENSE_RANK()` window function to assign ranks.
3. If multiple rows have the same score, they receive the same rank.
4. The next distinct score receives the next consecutive rank without any gaps.
5. Return the score along with its corresponding rank.

---

### ⏱️ Complexity

**Time Complexity:** `O(n log n)` *(sorting the scores before ranking)*

**Space Complexity:** `O(n)` *(used internally for sorting and window function processing)*
